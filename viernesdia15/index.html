Web Semántica

ELEMENTOS CON SIGNIFICADO!
Tipos de etiquetas:
  • Ejemplos de etiquetas NO semánticas: DIV, SPAN
  • Etiquetas Semánticas: IMG, FORM, TABLE, ARTICLE
Etiquetas SEMÁNTICAS PRINCIPALES:

Header

Definición: Cabecera de una sección o página
Semantic: Con esta sección identificamos dónde estará nuestro logo, nuestro título principal (

) y la barra de navegación. ETC...

Nav

Definición:Normalmente se coloca dentro del header. Son los elementos que nos permitirán movernos por el sitio WEB
Semantic: Con esta etiqueta identificamos las secciones que poseen enlaces de navegación, y facilitan a los motores de búsqueda y usuarios, movernos entre páginas. No todos los links tienen que estar dentro de un NAV.

Main

Crea un contenedor para el contenido principal

Section

Dónde se agrupan los contenidos relacionados entre sí. Normalmente incluye un título y una web se compone de varias secciones, cada una con su propio contenido y propósito. Es un apartado de la página web que agrupa un contenido relacionado. Normalmente está dentro del MAIN. Suelen incluir un título y una web se compone de varias secciones, cada una con su propio contenido y propósito
Ejemplos:
  • Capítulos
  • Introducción
  • Noticias
  • Productos
  • Un hero
  • Información de contacto

Article

Definición: Define un contenido independiente que puede existir por si mismo INFEPENDIENTE El mísmo puede ser compartido y utilizado en otros sitios o secciones
El article puede contener múltiples cosas no hay una respuesta correcta simplemente ayuda a los navegadores y a los usuarios a tratar de comprender nuestro código. Sirve para hacer un buen SEO, ya que si tenemos bien utilizada nuestra semántica WEB el navegador puede sacar ciertos elementos y hacer publicidad automática. Ejemplos de uso:
  • Un post de un blog
  • UN articulo de una revista
  • Comentario de usuario
  • Tarjeta de un producto

Aside

Contenido relacionado a nuestro contenido principal pero indirectamente. Se utiliza mucho para hacer publicidad
Ejemplos de USO:
  • Publicidad
  • Biografía de un autor
  • Enlaces relacionados
  • Información de perfil

Footer

Pie de página, sitio o sección. Pueden haber múltiples por página, pero no puede haber un footer dentro de otro, ni dentro de una header
Ejemplos de USO:
  • Contacto
  • Términos y condiciones
  • SiteMap (Permite ver todas las secciones de un sitio web, por niveles)(Sirve para ayudar a los buscadores a visitar al completo nuestra página WEB en un tiempo reducido)
  • Formulario de contacto
  • Mapa

Figure

Definir un elemento autocontenido como puede ser una imagen, un gráfico, una tabla etc..
Dentro de ella utilizaremos otra etiqueta Figcaption(Esta etiqueta describe el gráfico, imágen, tabla etc...EL TÍPICO PIE DE FOTO)

Mark

Resaltar contenido.Ssuele mostrar color amarillo.
