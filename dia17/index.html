Media Queries

¿Qué son las Media Queries (MQ)?

Declaración en CSS que nos permite definir como se va a visualizar nuestra web.

Sintaxis de MQ

Una MQ se compone de multiples parter:

  • Keyword: @media es una palabra clave, reservada
  • MediaType: Un tipo de medida. Ej: screen, print, all
  • Operador/es: EJ: and, not, only
  • Breakpoints: La condición a cumplirse. Ej: max-width, min-width
  • Reglas CSS: Bloque de reglas

Tipos de Media (Screen/Print)

- Print: Destinado a material de vista de impresión.
- Screen: Pantallas s color (Es el utilizado por defecto) - All: Todos los dispositivos
- Pueden encontrar otros tipos que ya no se usan: tv, handheld, speech, braille, e.t.c

Tips para utilizar MediaQueries

  • Los MQ sobreescriben las reglas de CSS anteriores, PERO no poseen mayor especificidad que las otras reglas por eso debemos agregarlas al final del documento
  • Deben utilizar los mismos selectores que quieran reemplazar
  • PENSAR BIEN LOS BREAKPOINTS
  • Podemos utilizar un @media

Mobile vs Tablet vs Desktop

- Mobile (55%): 320px - 768px
- Tablet (2%): 768px -1024px
- Desktop (43%): 1024px- 1920px o más

Uso de MQ en HTML

Podemos utilizar los @media directamente el el HTML utilizando el atributo "media". Antes que nada, debemos asegurarnos de que nuestro HTML incluya la etiqueta META en el head. meta name="viewport" content="width=device-width, initial-scale=1.0"
